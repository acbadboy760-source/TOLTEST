# Security Policy

## Purpose
This tool is intended **solely** for authorized security testing, educational research, and stress-testing your own infrastructure. Any use of this tool for unauthorized attacks is strictly prohibited.

## Reporting Misuse
If you witness misuse of this software, please open an issue in this repository or contact the maintainer directly.

## Responsible Development
We are committed to removing any content that facilitates illegal activity. This project is maintained as a proof-of-concept for defensive security research.
